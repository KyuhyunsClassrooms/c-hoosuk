# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
```
시작

플레이어 총합(playerTotal = 0)
딜러 총합(dealerTotal = 0)

반복 (사용자가 '0: 종료'를 선택할 때까지):
    메뉴 출력 (1: 게임 시작, 0: 종료)
    사용자 선택 입력

    만약 선택이 1이면:
        블랙잭 게임 실행 함수 호출

    만약 선택이 0이면:
        반복 종료

반복 끝

"게임 종료" 출력

종료


```

## 2. 핵심 기능 1: (예: 학생 등록 함수)
```
시작

    플레이어 총합을 0으로 초기화
    딜러 총합을 0으로 초기화

    ---- 플레이어 턴 ----
    반복:
        1~11 사이의 난수 카드 한 장 뽑아 playerTotal에 더함
        현재 플레이어 총합 출력

        만약 playerTotal > 21 이면:
            "플레이어 Bust! 패배" 출력
            함수 종료

        사용자에게 "카드를 더 뽑겠습니까? (1: 계속 / 0: 그만)" 입력받기
        만약 0이면 반복 종료

    ---- 딜러 턴 ----
    반복 (dealerTotal < 17):
        1~11 난수 카드 뽑아 dealerTotal에 더함
        현재 딜러 총합 출력

        만약 dealerTotal > 21 이면:
            "딜러 Bust! 플레이어 승리" 출력
            함수 종료

    ---- 승자 판정 ----
    만약 playerTotal > dealerTotal:
        "플레이어 승리" 출력

    만약 playerTotal < dealerTotal:
        "딜러 승리" 출력

    만약 playerTotal == dealerTotal:
        "무승부" 출력

종료


```
