# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
```
시작

전적 변수 초기화 (win = 0, lose = 0, draw = 0)

반복 (사용자가 '0: 종료'를 선택할 때까지):
    메뉴 출력 (1: 게임 시작, 0: 종료)
    사용자 선택 입력
    
    만약 선택이 1이면:
        가위바위보 게임 실행 함수 호출
    
    만약 선택이 0이면:
        반복 종료

반복 끝

전체 전적 출력 (승/패/무)

종료

```

## 2. 핵심 기능 1: (예: 학생 등록 함수)
```
시작
    사용자에게 선택 입력받기 (0: 가위, 1: 바위, 2: 보)

    만약 사용자 입력이 0~2 이외이면:
        "잘못된 입력" 출력
        함수 종료

    컴퓨터 선택을 난수(0~2)로 생성

    사용자 선택과 컴퓨터 선택 비교하여 승패 판정:
        만약 사용자 == 컴퓨터:
            "무승부" 출력
            draw 증가
        만약 (사용자 == 0 AND 컴퓨터 == 2) OR
             (사용자 == 1 AND 컴퓨터 == 0) OR
             (사용자 == 2 AND 컴퓨터 == 1):
            "사용자 승리" 출력
            win 증가
        위의 경우가 아니면:
            "사용자 패배" 출력
            lose 증가

    사용자 선택과 컴퓨터 선택 출력

종료

```
