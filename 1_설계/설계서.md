# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
가위바위보 게임 프로그램

## 2. 핵심 기능 목록
사용자에게 가위(0), 바위(1), 보(2) 중 하나 입력받기

컴퓨터가 난수(0~2)로 가위바위보 선택

사용자 선택과 컴퓨터 선택 비교

승패 판정 후 결과 출력

게임을 여러 번 반복할 수 있도록 구성

종료 시 전체 전적(승/패/무 승부) 출력


## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
char *menu[3] = {"가위", "바위", "보"};  // 선택 이름 저장

int win = 0;     // 승리 횟수
int lose = 0;    // 패배 횟수
int draw = 0;    // 무승부 횟수
```

**방법 2: 구조체 사용 (선택)**
```c
// 가위바위보 결과 저장용 구조체
struct Record {
    int win;
    int lose;
    int draw;
};

struct Record record = {0, 0, 0};

```
