# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
블랙잭 게임

## 2. 핵심 기능 목록
카드 값을 1~11 사이 난수로 생성

플레이어는 카드 추가 여부 선택

딜러는 17 미만이면 자동으로 카드 뽑기

21을 초과하면 Bust(패배)

21 이하에서 더 높은 점수가 이김

결과 출력


## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
int playerTotal = 0;   // 플레이어 카드 총합
int dealerTotal = 0;   // 딜러 카드 총합

int playerBust = 0;    // 플레이어가 21 초과 여부
int dealerBust = 0;    // 딜러가 21 초과 여부

```

**방법 2: 구조체 사용 (선택)**
```c
// 플레이어/딜러 정보 저장 구조체
struct Player {
    int total;     // 카드 총합
    int bust;      // Bust 여부 (1 = 21 초과)
};

// 두 명 생성
struct Player player = {0, 0};
struct Player dealer = {0, 0};

```
