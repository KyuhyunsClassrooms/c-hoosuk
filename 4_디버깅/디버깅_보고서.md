## 🐛 발견한 오류 #1

### 1) 어떤 문제가 발생했나요?
```
플레이어 카드 합이 25가 되었는데도 게임이 계속 진행되었습니다.
```

### 2) 오류가 발생한 코드
```c
if (playerTotal > 21);
{
    printf("Bust!\n");
}

```

### 3) 왜 오류가 생겼나요?

```
if () 뒤에 ; 을 붙여서 조건문이 무효가 됨

```

### 4) 어떻게 고쳤나요?
```c
if (playerTotal > 21) {
    printf("Bust!\n");
    return;
}

```

### 5) 고친 후 결과
- [o ] ✅ 잘 작동해요!

## 🐛 발견한 오류 #2

### 1) 어떤 문제가 발생했나요?
```
딜러가 17, 19, 21을 넘어도 무한히 카드를 뽑았습니다.

```

### 2) 오류가 발생한 코드
```c
while (dealerTotal <= 17) {
    dealerTotal += drawCard();
}

```

### 3) 왜 오류가 생겼나요?
```
딜러 규칙은 17 이상이 되면 멈춰야 하는데,
조건을 <= 17 로 써서 17에서도 한 번 더 뽑는 문제가 발생했습니다.
```

### 4) 어떻게 고쳤나요?
```c
while (dealerTotal < 17) {
    dealerTotal += drawCard();
}

```

### 5) 고친 후 결과
- [o] ✅ 잘 작동해요!

## 💭 디버깅하면서 배운 점

```
블랙잭 프로그램을 디버깅하면서, 작은 실수 하나가 결과에 큰 영향을 준다는 것을 배웠습니다. if 뒤 세미콜론, 난수 초기화, 반복문 조건 등 사소한 부분도 프로그램 흐름에 중요하다는 것을 느꼈습니다. 오류 메시지를 확인하고 결과를 관찰하는 것이 문제를 빠르게 찾는 데 큰 도움이 된다는 것도 알게 되었습니다.
```
